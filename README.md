# 🖥️ iOS Developer Profile (Design Only)

## 📸 Project Overview

This project showcases a sleek and modern iOS developer profile screen, designed purely for visual aesthetics and UI practice. It captures the clean, polished style of Apple’s design language, focusing on personal branding for developers.

<img src="https://github.com/user-attachments/assets/bc5ca880-5be8-4c3e-bfd1-26278aa27de0" width="600"  alt="Profile Preview">

## 🎨 Features

* Personalized profile picture with a circular frame
* Elegant background with a natural, architectural vibe
* Clean, bold typography for name and bio
* Minimalist SF Symbols representing key interests (technology, VR, books, fitness)
* Call to action button for visual balance (not functional)

## 🛠️ Tools Used

* Swift
* SwiftUI
* SF Symbols
* Local images for background and profile

## 🌟 Key Elements

* **Profile Picture**: Central, round frame for a personal touch
* **Background**: Modern architecture with warm tones for a professional yet welcoming feel
* **Typography**: Clean, sans-serif fonts for readability and impact
* **Iconography**: Colorful and minimalist icons to represent various interests
* **CTA Button**: Bold, blue button for immediate visual impact (not functional)

## 🚀 How to Use

This design is intended for inspiration and aesthetic practice. It is not a fully functional app. You can recreate this design in SwiftUI using local images and SF Symbols for a similar effect.

## 🖌️ Design Inspiration

The design aims to capture the spirit of innovation and personal connection that defines the Apple ecosystem, blending technology with personal storytelling.
